## 📖 About The Project

This portfolio serves as my digital space on the internet—a place where I share my projects, technical skills, work experience, and the technologies I'm currently learning and working with.

Rather than being just a resume website, I wanted it to reflect my continuous learning journey. The portfolio highlights my work across web development, data science, and software engineering while maintaining a clean and responsive user experience.

Whether you're a recruiter, developer, or fellow learner, I hope this gives you a glimpse into what I love building.

## ⚙️ Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
